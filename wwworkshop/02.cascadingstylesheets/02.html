Selector { property: value }

HTML

        <TAGNAME ATTRIBUTE="VALUE">text</TAGNAME>




        <p id="one">text</p>

CSS


        selector {
            property: value;
            otherproperty: value;
        }

Selector

Tag name: h1, h2 or p

        h1 {
            font-size: 20px;
        }

        p {
            font-size: 16px;
        }

Id: id value, preceded by #. For example:

        #one {
            color: blue;
        }

        #two {
            color: red;
        }

Class name, preceded by dot


        .class {}

        .another-class {}

Classes are like Ids, but don't have to be unique.


        HTML:

        <p class="intro red">An introductory text in red</p>



        CSS

        .intro {
            font-size: 20px;
        }

        .red {
            color: red;
        }

Properties

Some properties are:

        p {
            font-size: 16px;
            color: black;
            line-height: 1.4em;
            text-align: left;
            word-spacing:  -0.05em;
            text-indent: 1.4em;
            margin-bottom: 0;
            background-color: pink;
        }
